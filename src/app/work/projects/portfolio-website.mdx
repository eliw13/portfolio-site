## Overview

A personal portfolio website showcasing my work as an Applications Analyst and photographer. Built with modern web technologies and featuring real-time integrations with Discord and GitHub.

## Key Features

- **Real-time Discord Presence**: Live status updates showing current activities via Lanyard API
- **GitHub Integration**: Dynamic display of GitHub profile information
- **iOS-Inspired Glass UI**: Custom liquid glass navigation bar with chromatic aberration effects
- **Responsive Design**: Optimized for all screen sizes from mobile to desktop
- **Dark Mode**: Seamless dark mode support throughout the site

## Technologies Used

- **Framework**: Next.js 14 with TypeScript
- **Styling**: Once UI design system
- **APIs**: Discord (Lanyard), GitHub API
- **Deployment**: Vercel
- **Version Control**: Git & GitHub

## Implementation Highlights

### Discord Integration
Implemented real-time Discord presence using the Lanyard WebSocket API, displaying current status and activities with automatic updates every 30 seconds.

### Custom Glass Components
Created custom React components with SVG filters for advanced visual effects including:
- Backdrop blur and distortion
- RGB channel offsets for chromatic aberration
- Responsive scaling and animations

### Performance Optimization
Optimized for fast load times and smooth interactions through:
- Efficient component rendering
- Proper image optimization
- Strategic code splitting
